# Credit Card Accounting Skill

當開發信用卡記帳功能時需要使用此技能。

## When to use this skill

- 首先需要遵照 TDD 開發流程定義好開發規格
- 先建立前端 UI 來確定後續開發方向
- 使用 Vitest 以及 Supertest 建立單元測試
- 如果沒有信用卡 Model 需要先建立，且需要先發想 DB Schema
- 當有提到權限、登入、登出等規則時需要閱讀 auth_spec.md
- 當有提到交易、帳戶、類別等規則時需要閱讀 transaction_spec.md
- 當有提到使用者、通知等規則時需要閱讀 user_spec.md
- 當有提到統計、報表等規則時需要閱讀 statistics_spec.md
- 當有提到 Excel 檔案時需要閱讀 excel_spec.md
- 當有提到類別時需要閱讀 category_spec.md
- 當有提到通知時需要閱讀 notification_spec.md
